Доступность: 6/10, для доступа к кодингу необходима регистрация (для создания private repls необходимо платить)
Пользовательский интерфейс: 8/10, интерфейс довольно масштабный, без русской локали, но быстро разобраться в самом минимальном функционале можно
Возможности символьной отладки на уровне исходных текстов: 10/10 присутствует (???)
Отображение информации о регистрах: 10/10, отображается
Простота использования: 10/10, легко компилирует код, поддерживает отступы при его написании, можно загрузить код из файла или скопировать и вставить
Наличие информации для изучения: 7/10, показывает вывод в консоль (если тот предусмотрен программой), значения переменных на breakpoint-х, при завершении программы затраченные время и память не выводит
Отдельные плюсы: подсветка синтаксиса присутствует, можно изменить размер шрифта, трансформировать окно дебага, есть возможность поделиться проектом с другими пользователями, также есть интеграция с GitHub
Отдельный минус: для написания кода на ассемблере необходимо создать программу на Си (сам ассемблер в списке выбрать нельзя), затем загрузить в проект файлы .s, запустить встроенный GDB и перевести их в исполняемые файлы 
Общая субъективная оценка: 8/10, многофункциональная среда для совместной работы с кодом, поддерживает много ЯП, в качестве дебага используется встроенный GDB (TUI), довольно жадная
