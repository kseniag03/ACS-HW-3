Доступность: 9/10, можно запустить на Ubuntu
Пользовательский интерфейс: 7/10, простой интерфейс командной строки с дополнительными окнами для отображения всего кода и, например, регистров, удобнее, чем GDB, однако нельзя прокрутить команды терминала (неудобно для создания скриншотов, где будут видны все команды)
Возможности символьной отладки на уровне исходных текстов: 10/10, присутствует (???)
Отображение информации о регистрах: 10/10, отображается
Простота использования: 7/10, для запуска иного кода необходимо выйти из режима GDB, при активации TUI пропадает возможность выделять элементы для копирования (но работают горячие клавиши)
Наличие информации для изучения: 7/10, показывает вывод в консоль (если тот предусмотрен программой), значения переменных на breakpoint-х, при завершении программы затраченные время и память не выводит
Отдельные плюсы: подсветка синтаксиса присутствует, можно просмотреть весь код с указанием строк
Общая субъективная оценка: 8/10, удобный отладчик для освоивших командную строку, добавляет наглядности GDB
